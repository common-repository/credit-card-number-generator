Generate and Validate dummy credit card numbers for all well-known card issuers like MasterCard, Visa, JCB, Discover, American Express, etc.

== Description ==
Test your Payment Gateway Integration with our Test Credit Card Number generator and validator plugin. Payment gateway developers use our program for dummy credit card number generation and validation. Using this plugin you can have test card number generator and validator on your website and provide your visitors to get test card numbers.

These numbers DO NOT work in real transactions! They are for testing purposes only. Use it with CVV as 123 and future expiration dates.

Our plugin uses Luhn algorithm to generate and validate dummy card numbers. This algorithm is freely available across the web even on Wikipedia.org. So go ahead and install this plugin and enable your site visitors to generate / validate dummy card numbers from your website.

== Key Features ==

- Admin can enable/disable the plugin functionality.

- Admin can change button label and button text and background color.

- Use shortcodes to display generator and validator anywhere in posts and pages.

- You can even use it as a Widget.

== Installation ==

1. Upload 'credit-card-number-generator' to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' screen in WordPress admin dashborad.

3. Use the Settings > Credit Card Number Generator screen to configure the plugin setting.
